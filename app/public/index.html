Principales Sistemas Operativos

Resumen rápido de las plataformas más utilizadas en escritorio y servidores para que puedas justificar por qué tu app corre de forma uniforme en Ubuntu, Debian y Rocky Linux.

Objetivo: desplegar una sola app Node.js detrás de Nginx y mantener el mismo stack en cualquier VM del curso.

Conoce los puntos fuertes de cada sistema antes de decidir dónde correr tus servicios.

Linux

Kernel abierto, modular y estable. Base de la mayoría de los servidores en la nube.

  • Distribuciones populares: Ubuntu, Debian, Rocky Linux
  • Gestores de paquetes: `apt`, `dnf`, `pacman`
  • Ideal para contenedores, automatización y DevOps

Windows

Dominante en escritorio y gaming. Integración profunda con Active Directory y .NET.

  • Entorno gráfico familiar para usuarios finales
  • Soporte nativo para herramientas Microsoft
  • WSL permite trabajar con herramientas Linux

macOS

Basado en BSD con interfaz pulida. Preferido por desarrolladores móviles y creativos.

  • Terminal Unix y herramientas de desarrollo preinstaladas
  • Optimizado para el hardware de Apple
  • Excelente para crear contenido y apps iOS/macOS

Distribuciones para servidores

Ubuntu Server, Debian y Rocky Linux priorizan estabilidad y soporte prolongado.

  • LTS con parches de seguridad frecuentes
  • Administración remota sencilla por SSH
  • Compatibles con herramientas de automatización

Comparativa rápida

Sistema Fortalezas Casos de uso
Ubuntu/Debian Actualizaciones constantes, enorme comunidad Servidores web, aprendizaje, despliegues rápidos
Rocky Linux Clon de RHEL, ciclo estable, soporte empresarial Bases de datos críticas, entornos corporativos
Windows Compatibilidad con software propietario Escritorio, .NET, integración con AD
macOS Experiencia premium, herramientas creativas Diseño, desarrollo de apps iOS/macOS
